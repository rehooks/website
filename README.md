# Rehooks website

The official website for [Rehooks](https://rehooks.com)

## Contribution

If you need to make any changes to the website:

1. Clone this repo
2. Run `yarn` to install dependencies
3. Run `yarn dev` to start the server on `http://localhost:3000`
